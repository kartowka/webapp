## Getting Started

```bash
# Clone this repository
$ git clone https://github.com/kartowka/webapp.git

# Go into the repository
$ cd webapp

# Install dependencies
$ npm install

# Compile the code using typescript compiler
$ npm run build

# Run the app
$ npm run dev

#run the tests
$ npm run test
```
